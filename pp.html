My Projects

Smart Dustbin

A smart IoT-based dustbin system that uses an ultrasonic sensor to detect if the bin is full and a DHT sensor to monitor temperature and humidity. Alerts are sent to the server via MQTT using ESP32 (BharathPi), and the data is displayed on a dashboard in real-time.

Tech Used: ESP32, MQTT, Node.js, HTML, DHT Sensor, Ultrasonic Sensor.

Employee Database System

Forest Fire Detection and Protection using WSN (working)

This project uses a wireless sensor network to monitor forest temperature and smoke levels. When thresholds are crossed, alerts are sent to authorities. It uses ESP32 for real-time Wi-Fi communication, sensors for data collection, and a dashboard for visualization.

Tech Used: ESP32, Sensors, Wi-Fi, WSN, Firebase, HTML Dashboard.
